# 莉莉丝助手 (Lilith Assistant) - SillyTavern 插件

![Version](https://img.shields.io/badge/version-3.0.3--Fluid-bd00ff)
![SillyTavern](https://img.shields.io/badge/SillyTavern-Extension-blue)

一个为人格化助手“莉莉丝”设计的 SillyTavern (酒馆) 扩展组件。本项目已全面升级至 **v3.0.3 正式版**。

> **v3.0.3 更新内容**：
> 1. **✨ 非侵入式数值通知 (Toast)**：修复了随机事件触发时，好感度/理智变动弹窗会强行关闭并覆盖事件长文本的问题。现在所有数值变动（及 FP 变化）将以浮动通知的形式在头像上方飘过，不再打断阅读。
> 2. **🎨 视觉体验优化**：新增了特殊事件气泡的专属配色（紫色半透明），并优化了数值飘字的动画曲线。

> **v3.0.2 核心回顾**：
> 1. **💾 存储路径服务器化**：彻底摒弃浏览器 `localStorage`，所有数据统一存储至酒馆服务端。
> 2. **⚡ 体验平滑化**：移除所有更新确认弹窗，实现真正的“一键秒级更新”。

> **v3.0.0 核心回顾 (大脑皮层重构)**：
> 1. **🧠 莉莉丝的大脑皮层 (Dynamic Content)**：莉莉丝现在拥有了真实的“后台思考”能力。她会根据当前好感度、理智值及聊天背景，在后台自动构思多条专属对话与特殊事件，并在合适的时机主动插话。
> 2. **🎭 智能插话触发逻辑**：引入了基于心跳频率的归一化触发算法。100% 触发概率意味着莉莉丝将努力在下一个构思周期前，把她想对你说的所有话（及特殊事件）全部展示一遍。
> 3. **🛡️ 剧透保护机制**：AI 生成的构思内容在触发前会以“快照”摘要形式锁定在功能区页签，防止用户提前窥视即将发生的剧情。
> 4. **📱 移动端与 UI 精度优化**：
>    - 气泡定位算法重构，利用 CSS `calc` 实现自动避障，不遮挡头像且适配刘海屏/地址栏。
>    - 设置菜单重组，API 预设优先级提升。
>    - 增加了详细的中文配置说明与生成建议。

## 🌟 核心功能

- **🎭 五重人格切换**：毒舌魅魔、温柔人妻、雌小鬼、网络神人、柔弱妹妹。
- **📊 动态数值系统**：内置好感度 (Favorability) 与 理智值 (Sanity)，根据聊天内容动态增减。
- **🛡️ 智能正则盾**：支持自定义正则规则，自动过滤 AI 回复中的冗余内容或特定的剧情标签。
- **🎲 赌狗抽卡系统**：消耗 FP 点数从垃圾堆到神迹品质的随机道具获取。
- **🧠 记忆压缩归档**：自动对长对话进行 AI 总结并归档至“记忆碎片”，减少上下文压力。
- **✨ 模块化架构**：核心逻辑高度解耦，响应迅速，支持多种 API 协议 (OpenAI/Gemini)。

## 📂 插件结构

重构后的插件采用分层模块化设计，方便开发者进行二次开发或自定义：

```text
lilith-assistant/
├── index.js                # 插件入口，负责系统引导与启动
├── manifest.json           # 插件元数据配置
├── style.css               # UI 样式表
├── modules/                # 核心功能模块
│   ├── config.js           # 集中存放常量、配置及人格定义 (PERSONA_DB)
│   ├── storage.js          # 持久化存储，负责与酒馆 extensionSettings 同步
│   ├── assistant_manager.js# 逻辑中枢 (API 调用、心跳、抽卡、任务系统)
│   ├── ui_manager.js       # 渲染引擎 (HTML 生成、立绘管理、消息格式化)
│   ├── events.js           # 系统钩子 (监听酒馆消息渲染、发送前过滤等)
│   ├── audio.js            # 语音合成与播报
│   ├── persona.js          # 动态人格生成逻辑
│   └── utils.js            # 共享工具函数
└── assets/                 # 静态资源 (头像包、导出数据等)
```

## 🚀 安装方法

1.  打开 SillyTavern (酒馆)。
2.  进入 **扩展菜单 (Extensions)** -> **安装新扩展 (Install New Extension)**。
3.  在 URL 框中输入：`https://github.com/wt7141789/lilith-assistant`
4.  点击 **安装 (Install)**。
5.  在扩展设置中启用插件。

## ⚙️ 配置说明

插件支持三种连接模式：
- **酒馆内核 (推荐)**：使用酒馆当前的 API 设置。
- **OpenAI 兼容接口**：自定义 API Key 和 Endpoint。
- **Google Native**：直接连接 Gemini 接口。
